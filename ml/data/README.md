# Deepfake Datasets (Source & Processed)

This directory serves as the centralized repository for all datasets used during the training and validation of the machine learning models.

### 📂 Directory Structure
- **`audio/` & `images/` & `videos/`**: Raw source files downloaded from various benchmarks.
- **`processed_audio/` & `processed_videos/`**: Optimized tensors (.pt) generated by the preprocessing scripts (`ml/preprocess_*.py`). These are the actual samples used for training.

### 📊 Datasets Used

The models are trained on the following comprehensive real-world benchmarks:

#### 1. Audio
- **Audio Deepfake Detection Dataset**: High-quality real and synthetic voice samples.
  [Kaggle Link](https://www.kaggle.com/datasets/adarshsingh0903/audio-deepfake-detection-dataset?select=real_samples)

#### 2. Video
- **FaceForensics++**: Extracted frames from various manipulation techniques (Deepfakes, FaceSwap, etc.).
  [Kaggle Link](https://www.kaggle.com/datasets/adham7elmy/faceforencispp-extracted-frames?select=fake)
- **Celeb-DF v2**: Large-scale deepfake video dataset with refined visual artifacts.
  [Kaggle Link](https://www.kaggle.com/datasets/reubensuju/celeb-df-v2)

#### 3. Images
- **Deepfake VS Real 60k**: Balanced dataset for early backbone training.
  [Kaggle Link](https://www.kaggle.com/datasets/prithivsakthiur/deepfake-vs-real-60k?select=deepfake-vs-real-60k)
- **140k Real and Fake Faces**: High-resolution face datasets for model fine-tuning.
  [Kaggle Link](https://www.kaggle.com/datasets/xhlulu/140k-real-and-fake-faces)

### ⚠️ Note on Git
This directory is heavily restricted via `.gitignore`. The datasets contain millions of files and tens of gigabytes of data, which are **too large to commit** to the Git repository. Only the preprocessing scripts and directory structure are tracked.
