# Legacy Uploads Directory

This directory was used in earlier versions of the platform for storing temporary file uploads.

### 📂 Purpose
It is currently a **legacy/redundant** folder. The active system utilizes the hidden `.uploads` directory for better security and organization.

### ⚠️ Note on Git
This folder is ignored by `.gitignore` because the files processed by the system (Video, Image, Audio) are too large for Git version control and contain temporary user data.
